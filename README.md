# NFT 바우처 시스템

블록체인 기반 NFT 바우처 발급/사용/관리 플랫폼

## 팀 구성

| 이름 | 역할 | 담당 폴더 |
|------|------|-----------|
| 조익준 | 프론트엔드 | `frontend/` |
| 정재현 | 백엔드 | `backend/` |
| 조현 | 스마트 컨트랙트 | `blockchain/` |
| 황종훈 | 블록체인 | `blockchain/` |

## 레퍼런스 프로젝트

`../ttocket` — NFT 공연 티켓 서비스. 우리 프로젝트의 구조적 레퍼런스.

- **블록체인**: Solidity, Truffle, Ganache
- **프론트**: React, TypeScript, Web3.js
- **백엔드**: Spring Boot, MySQL, Redis
- **배포**: Docker, Nginx
