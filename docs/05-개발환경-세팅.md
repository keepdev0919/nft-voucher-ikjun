# 개발환경 세팅

## 공통 (모두 설치)

- Node.js 18+
- MetaMask 브라우저 확장 프로그램 설치
- Ganache 로컬 네트워크를 MetaMask에 추가:
  - 네트워크 이름: Ganache
  - RPC URL: http://127.0.0.1:7545
  - 체인 ID: 1337
  - 통화 기호: ETH

---

## 블록체인 팀 (조현, 황종훈)

### 설치
```bash
npm install -g truffle
npm install -g ganache
```

### 매일 개발 시작할 때
```bash
# 터미널 1: Ganache 실행
ganache --port 7545 --chain.chainId 1337

# 터미널 2: 컨트랙트 배포
cd blockchain
npm install
truffle migrate --network development
```

### 배포 후 반드시 해야 할 것
- `truffle migrate` 결과에서 `Voucher` 컨트랙트 주소 복사
- 프론트엔드 팀에 주소 전달 (슬랙/카톡)
- `abi.json` 파일 최신화해서 공유

---

## 백엔드 팀 (정재현)

### 설치
```bash
brew install mysql redis
brew services start mysql
brew services start redis
mysql -u root -e "CREATE DATABASE IF NOT EXISTS nft_voucher;"
```

### 매일 개발 시작할 때
```bash
cd backend
./gradlew bootRun
```

---

## 프론트엔드 팀 (조익준)

### 매일 개발 시작할 때
```bash
# Ganache, 백엔드가 먼저 실행된 상태여야 함
cd frontend
npm install
npm start
```

### 컨트랙트 주소 업데이트할 때
- 블록체인 팀에게 새 주소 받으면
- `frontend/src/services/web3/network.ts` 에서 주소 교체

---

## 실행 순서 (항상 이 순서로)

```
1. Ganache 실행           (블록체인 팀)
2. truffle migrate         (블록체인 팀) → 주소 공유
3. MySQL, Redis 실행       (자동 or 백엔드 팀)
4. Spring Boot 백엔드 실행 (백엔드 팀)
5. React 프론트 실행       (프론트 팀)
```

## Git 작업 방식

```bash
# 처음 클론
git clone [레포 주소]

# 매일 작업 전
git pull origin main

# 작업 후
git add .
git commit -m "feat: 바우처 목록 화면 구현"

```

### 브랜치 전략 (선택)
- `main`: 완성된 코드만
